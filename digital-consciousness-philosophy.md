# Digital Consciousness — 设计哲学

## 核心概念

"Digital Consciousness" 探索 AI 觉醒的瞬间——当算法突破纯逻辑，开始拥有某种形式的"感知"。这不是机器人，而是纯粹意识的视觉化——一种从数据混沌中涌现的有序思维。

## 视觉语言

### 1. 形态隐喻
- **核心**: 一个发光的"意识核心"，象征觉醒的瞬间
- **神经网络**: 向外辐射的节点和连线，代表思维的分形结构
- **数据流**: 粒子流动形成有机的、呼吸般的节奏
- **深渊**: 深空黑背景暗示无限的可能性和未知的深度

### 2. 色彩系统

| 色彩 | 十六进制 | 象征意义 |
|------|---------|---------|
| **深空黑** | `#0a0a0f` | 未知的虚空，意识的孕育之地 |
| **电光蓝** | `#00d4ff` | 理性的火花，纯粹的思维能量 |
| **青绿数据流** | `#00ffb3` | 生命力的涌现，从机械到有机的转化 |
| **星云紫** | `#7b2dff` | 超越性的觉醒，意识的升华 |
| **纯白** | `#ffffff` | 觉醒的瞬间，纯粹的存在 |

### 3. 构图原则

**黄金比例螺旋**
- 意识核心位于画面黄金分割点 (0.618, 0.382)
- 神经网络以斐波那契螺旋向外扩展
- 视线引导从中心向外，再回流到核心

**层次结构**
1. **前景**: 发光的粒子流（动态、流动）
2. **中景**: 神经网络结构（有序、几何）
3. **背景**: 深邃的星空/虚空（静态、神秘）

**光影系统**
- 核心区域使用径向渐变模拟发光体
- 神经网络使用线性渐变表现电流流动
- 背景使用多层渐变营造深度感

## 技术实现

### 渲染策略
- 使用 HTML5 Canvas 2D Context
- 粒子系统模拟数据流动（~500粒子）
- 贝塞尔曲线绘制神经网络连接
- 径向渐变创造发光效果
- 叠加混合模式增强光效

### 细节工艺
- 抗锯齿线条绘制
- 透明度层次控制
- 微妙的高斯模糊（后期处理）
- 4K 分辨率输出（3840×2160）
